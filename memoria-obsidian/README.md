# Memoria do Jarvis

Abra esta pasta como um vault no Obsidian.

Esta memoria guarda o contexto de longo prazo do Jarvis em arquivos Markdown simples. A ideia e manter tudo local, legivel e facil de editar.

## Estrutura

- `00 Perfil do Chefe.md`: identidade, tom, limites e preferencias.
- `01 Projetos.md`: objetivos ativos e contexto dos projetos.
- `02 Tarefas.md`: proximas acoes, pendencias e tarefas recorrentes.
- `03 Historico resumido.md`: resumos de conversas importantes.
- `04 Aprendizados do Jarvis.md`: padroes percebidos sobre o jeito do Chefe.

## Como atualizar

Na interface do Jarvis, use o comando:

```text
/obsidian
```

Ele copia um resumo em Markdown para a area de transferencia. Depois, cole o conteudo na nota adequada.
